# Deprecated alias for [`audit()`](https://bbuchsbaum.github.io/mixeff/reference/audit.md)

`audit_design()` and
[`audit()`](https://bbuchsbaum.github.io/mixeff/reference/audit.md) were
two names for one operation; the surface collapsed to
[`audit()`](https://bbuchsbaum.github.io/mixeff/reference/audit.md) for
0.2.0. This alias forwards with a deprecation note and will be removed
in a future release.

## Usage

``` r
audit_design(spec)
```

## Arguments

- spec:

  An `mm_spec` or `mm_fit`; see
  [`audit()`](https://bbuchsbaum.github.io/mixeff/reference/audit.md).

## Value

See [`audit()`](https://bbuchsbaum.github.io/mixeff/reference/audit.md).
